# سامانه مدیریت مقالات و رویدادهای علمی

این پروژه یک سامانه تحت وب برای مدیریت مقالات، اخبار و رویدادهای علمی است.

## امکانات

- ثبت نام و ورود اعضا
- ارسال مقالات و اخبار توسط اعضا
- ایجاد رویدادهای علمی توسط مدیر
- تایید و انتشار محتوay ارسالی توسط مدیر
- داشبرد کاربری برای اعضا

## نقش‌ها

- **مدیر (Admin):** توسط اسکریپت پایتون ساخته می‌شود و مسئولیت تایید محتوا را بر عهده دارد.
- **عضو انجمن (Member):** می‌تواند در سایت ثبت نام کرده و مقالات و اخبار خود را برای تایید ارسال کند.

## تکنولوژی‌های مورد استفاده

- **Backend:** FastAPI
- **Frontend:** HTML, CSS, JavaScript
- **Database:** PostgreSQL

## راه اندازی و نصب

برای راه اندازی این پروژه، مراحل زیر را دنبال کنید:

1.  **کلون کردن پروژه:**
    ```bash
    git clone <repository-url>
    cd <project-directory>
    ```

2.  **ساختن و فعال کردن محیط مجازی:**
    ```bash
    python -m venv venv
    # On Windows
    # venv\Scripts\activate
    # On macOS/Linux
    source venv/bin/activate
    ```

3.  **نصب وابستگی‌ها:**
    ```bash
    pip install -r requirements.txt
    ```

## اجرای برنامه

پس از نصب وابستگی‌ها، می‌توانید برنامه را با دستور زیر از مسیر اصلی پروژه اجرا کنید:

```bash
uvicorn app.main:app --reload
```

برنامه بر روی آدرس `http://127.0.0.1:8000` در دسترس خواهد بود.

### صفحات موجود

-   **صفحه اصلی:** `http://127.0.0.1:8000/`
-   **صفحه ثبت نام:** `http://127.0.0.1:8000/register`
-   **صفحه ورود:** `http://127.0.0.1:8000/login`
-   **داشبرد:** `http://127.0.0.1:8000/dashboard`
